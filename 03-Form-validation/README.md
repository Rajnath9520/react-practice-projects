## React Registration Form

A simple and clean Registration Form built using React Hooks.
This project demonstrates controlled inputs, form validation, conditional UI states, and state-based rendering.

# Features

Controlled form inputs using useState
Real-time validation for empty fields
Error message handling
Disabled submit button when form is incomplete
Stores and displays registered users
Clean UI with Tailwind CSS classes

# Concepts Used

React Functional Components
useState Hook
Controlled Inputs
Event Handling (onChange, onSubmit)
Conditional Rendering
Array State Updates with Spread Operator
Basic Form Validation
